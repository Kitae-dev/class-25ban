# 🚀 숙명여중 2-5반 학급 앱 배포 가이드

## 학부모 앱과 동일한 Supabase 프로젝트 사용 가능
※ 테이블 이름이 class_ 접두어로 구분되므로 같은 Supabase 프로젝트 사용 OK

## STEP 1. Supabase DB 구성
1. Supabase → SQL Editor
2. supabase/schema.sql 전체 붙여넣기 → Run

## STEP 2. 환경변수 설정
.env.example → .env 복사 후 실제 값 입력

## STEP 3. 로컬 테스트
npm install
npm run dev
→ http://localhost:5173

## STEP 4. GitHub 업로드
git init
git add .
git commit -m "학급 앱 초기 배포"
git remote add origin https://github.com/[아이디]/class-25ban.git
git push -u origin main

## STEP 5. Vercel 배포
1. vercel.com → New Project → class-25ban 선택
2. Environment Variables 설정:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
   - VITE_ADMIN_PASSWORD
3. Deploy

## URL 구조
- 학생용:  https://class-25ban.vercel.app
- 반장용:  https://class-25ban.vercel.app/admin/login

## 반장 관리 기능
- 공지 작성/수정/삭제 (카테고리별)
- 투표 생성/마감/삭제 (단일/복수 선택)
- 댓글 삭제
